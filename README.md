# ETF 智能分析系统

这是一个基于 Python 的 ETF 智能分析工具，用于实时分析 ETF，输出评分和操作等级（强烈买入 / 买入 / 偏多持有 / 中性观望 / 卖出 / 强烈卖出）。

> 该工具支持 AI 动态权重与动态参数优化，并融合了 TMSV 复合指标作为内部因子。若未配置 AI Key，则使用默认权重和参数。

## 🚀 最新优化（基于7天历史数据）

- **历史数据保存**：`etf_state.json` 现在保存7个交易日的评分历史
- **智能趋势分析**：利用7天数据计算分数趋势斜率，识别上涨/下跌周期
- **动态参数调整**：基于历史表现自动调整买入/卖出阈值和确认天数
- **高波动适应**：在高波动环境下自动切换到更保守的策略
- **反弹与冲高机制**：识别短期反弹机会和冲高卖出时机

## 📁 文件说明

- `main.py` - 主程序入口，负责获取数据、计算指标、生成评分并输出结果。
- `positions.csv` - ETF 标的列表，必须包含 `代码` 和 `名称` 两列。
- `etf_state.json` - 运行时保存的状态文件，用于记录历史评分和确认逻辑。
- `weight_cache.json` - AI 权重/参数缓存文件。
- `data_cache.json` - 可选缓存数据文件（当前脚本未直接引用）。

## ✅ 运行环境要求

需要 Python 3.8+，并安装以下依赖库：

- `baostock`
- `requests`
- `numpy`
- `pandas`
- `openai`

如果使用虚拟环境，请在仓库根目录中创建并激活：

```bash
python -m venv .venv
.\.venv\Scripts\activate
```

安装依赖：

```bash
pip install baostock requests numpy pandas openai
```

## ⚙️ 配置说明

### 1. ETF 标的列表

`positions.csv` 需要包含至少以下两列：

- `代码` - ETF 代码，例如 `sh.510300`
- `名称` - ETF 名称

示例：

```csv
代码,名称
sh.510300,沪深300ETF
sh.510500,中证500ETF
```

### 2. AI Key（可选）

如果要启用 AI 动态权重与参数调整，请在环境变量中配置：

```powershell
setx DEEPSEEK_API_KEY "your_api_key"
```

重启命令行后生效。若不配置该变量，脚本将使用默认权重和默认参数，并采用简单规则判断市场状态。

## ▶️ 使用方法

在仓库根目录执行：

```bash
python main.py
```

脚本执行后将输出如下字段：

- `名称`
- `代码`
- `价格`
- `评分`
- `操作`

输出结果示例：

```text
名称             代码           价格   评分 操作
------------------------------------------------------------
沪深300ETF       sh.510300   2.345  0.52  买入
```

## 🧠 算法说明

脚本核心功能包括：

- 从 `baostock` 获取历史日线和周线数据
- 计算常见技术指标：MA、MACD、KDJ、布林带、威廉指标、RSI、ATR 等
- 生成 TMSV 复合指标用于评分体系内部参考
- 根据买卖权重计算买入/卖出信号
- 使用 `score_history` 进行多日确认逻辑
- 根据评分输出操作等级

## 📌 行为说明

- 默认权重文件:
  - 买入权重由 `DEFAULT_BUY_WEIGHTS` 控制
  - 卖出权重由 `DEFAULT_SELL_WEIGHTS` 控制
- 默认参数由 `DEFAULT_PARAMS` 控制
- 当设置 `DEEPSEEK_API_KEY` 时，可通过 AI 接口动态生成权重和参数
- `etf_state.json` 用于保存历史评分，支持连续确认天数和风险提示

## 💡 注意事项

- 请确保 `positions.csv` 内容格式正确；若文件缺失或格式异常，脚本会报错。
- 若出现 `baostock` 登录或数据获取失败，脚本会直接退出。
- 若未配置 AI Key，则不会使用 `deepseek` 接口，仅使用内置策略。
